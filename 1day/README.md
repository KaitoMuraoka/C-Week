# 最も基本的なプログラム

## エスケープシーケンス
`\n` は改行を表す特殊な文字。
この `\` で始まる文字れを、**エスケープシーケンス**という。

| 記号 | 意味 |
| ---- | ---- |
| \a | 警告音 |
| \b | バックスペース |
| \n | 改行 |
| \t | タブ |
| \\ | 文字列としての `\` |
| \? | 文字列としての ? |
| \" | 文字列としての " |
| /' | 文字列としての ' |
| \0 | null の文字列 |

## 書式設定
| 記号 | 意味 | 使用例 |
| --- | --- | ---|
|　%d | 整数値を10進数で表示する | 1, 12, 30, -4, 5 + 5 |
| %f | 実数値を10進数で表示する | | 
| %lf | 実数値を10進数で表示する(&f より長い桁が表示可能) | |
| %c | 文字、ASCⅡコードで表示された文字列が表示される | 'A', 'b', '8' |
| %s | 文字列、文字列をそのまま表示できる. | "ABC", "田中太郎", "Tarou Tanaka" |

また、%d や、%f は数値だけでなく、計算式を用いても良い
